Det er en forenklet selvbetjeneste Renting System, hvor man kan låne forskjellige kjøretoy.
Programmet kjøres i to forskjellige mode - admin mode og user(kunde) mode.
Hver mode har forskjellige interface og funksjoner (features). Du kan prøve selv.
I person.txt lagres det informasjon om brukere - 'a' identifikerer admin, 'k' - kunde.
Logging skjer med fodselsnummer. Titt på person.txt og logg inn både som admin og kunde.
Du kan også være ny kunde, prøv det også.

For å kjøre programmet må du ha Java Development Kit (JDK) instalert
Skriv kommand i cmd: 
\DinPath> java -jar RentingSystem.jar
